# Avatar Journey - 虚拟旅行助手 (华欣版集成)

一个基于 AI 驱动的沉浸式虚拟旅行角色扮演应用，通过先进的 **Google Gemini** 模型实现文字生成与虚拟地理坐标同步。

## 项目概述

Avatar Journey 采用极具张力的 **粗野主义 (Brutalist)** 设计风格，在桌面端模拟了全功能的 **iPhone** 交互界面。应用通过人工智能实现了一个智能虚拟化身，它会定期生成写实的旅行事件，并记录下精确的经纬度轨迹。

**注：当前版本已全面集成“华欣版”核心架构与虚拟 GPS 轨迹系统。**

## 核心特性

- **粗野主义美学**：黑白高对比度设计，大胆的排版与边框，模拟 430px 宽度的真实 iPhone 交互框架。
- **分屏布局**：左侧展示系统说明与玩法，右侧沉浸式操作虚拟手机。
- **地图轨迹系统 (Trajectory)**：新增了基于虚拟经纬度（GPS）的实时足迹追踪。每一天的旅行记录、观测终端都将同步显示化身的移动轨迹，记录下每一公里走过的路程。
- **AI 核心驱动 (Gemini)**：
  - **文本生成**：使用 Gemini 1.5 Flash 模型，生成如文学作品般生动的旅行情节。支持高度拟事、平实自然的游记文风。
  - **地理逻辑**：严格限制角色的移动范围，确保经纬度坐标与当前所在的城市（如华欣）完美契合，杜绝地理幻觉。
  - **离线补全**：延迟检测并自动补全失联期间的历程。
- **数值与轨迹深度联动**：
  - **观测终端**：数值页面新增了当日轨迹地图，让“心情、能量、经验”等数值拥有了地理实存感。
  - **双币系统**：人民币 (CNY) 与 泰铢 (THB) 实时演算。
- **档案室地图功能**：在“旅行档案室”中翻阅过往日志时，每一篇日记底部都会自动调取并渲染当天的路线图。
- **全新扩展系统** (从华欣版迁移)：
  - **衣柜系统 (Wardrobe)**：管理虚拟化身的实时穿搭与行李装备，影响差旅感知。
  - **人际关系 (Social)**：多维度的社交关系追踪，记录与特定 NPC（如 Krit）的互动历史、亲密度与关键突破点。
- **自动日记捕获**：系统自动感应时区与日期变化，在午夜前后或重启应用时自动完成前日总结。
- **离线导出手稿**：支持一键将所有旅行信号导出为 Markdown 格式的离线手稿。
- **多端持久化**：采用 **IndexedDB** 进行大规模结构化数据存储，支持通过 JSON 文件进行存档的导入与导出。

## 技术栈

- **前端架构**：React 18 (Vite) + TypeScript
- **设计系统**：Vanilla CSS (Brutalist Design Tokens) + Lucide Icons
- **AI 赋能**：字节跳动豆包 (Doubao) API / Claude (通过迁移逻辑支持)
- **存储**：IndexedDB (AvatarJourneyDB) + 华欣版 JSON 状态快照注入

## 开始使用

1. **终端运行**：`npm run dev` 启动本地实验室环境。
2. **状态注入**：系统启动时会自动检测并注入“华欣版”角色“阿葡”的记忆与档案。
3. **衣柜管理**：在手机界面进入“衣柜”分页查看当前穿搭。
4. **关系追踪**：在“关系”分页查看与 NPC 的最新互动动态。

## 最新更新 (v6)

- **数据存储重构**：升级至 v6 架构，优化了社交人物（NPC）的顶层属性，确保 personality 等关键字段被精确渲染。
- **社交共鸣增强**：
  - **浪漫关系识别**：系统自动将 `relationType: romantic` 的联系人渲染为爱心评分（心脏填充逻辑），`friend` 则渲染为星号。
  - **黑白暴力美学**：移除了所有彩色填充，统一为极致的黑/白对比色，符合粗野主义标准。
- **JSON 全量驱动**：整个应用（包含坐标、穿搭、行李、关系网等）均通过标准 JSON 协议与 AI 联动，支持高度自定义的剧情推进。
- **多存档管理 (SLOTS)**：在“角色设定”页新增历程封存台，支持在多个平行旅行状态间无缝切换。
- **视觉层级优化**：
  - **社交页面**：大幅拉开章节间距，消除活跃关系与历史存档的视觉重叠。
  - **设置终端**：统一了全局按钮的粗野主义风格，增强了物理投影感与交互反馈。
  - **核心人设锁定**：增加了旅程进行中的档案锁定机制，确保角色一致性。
