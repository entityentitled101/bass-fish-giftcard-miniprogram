# 部署指南

## 推送到 GitHub

### 1. 添加项目到 Git

在「鲈鱼兑奖」文件夹下执行：

```bash
# 添加项目到 Git
git add gift-card-miniprogram/

# 提交
git commit -m "feat: 添加礼品卡小程序 - 3D展示、领取、兑奖功能"

# 推送到 GitHub
git push origin main
```

### 2. 敏感信息安全

以下文件已被 `.gitignore` 保护，**不会**提交到 GitHub：
- ✅ `config.js` - 包含 API Keys
- ✅ `.env` - 环境变量
- ✅ `project.private.config.json` - 微信私有配置

提交到 GitHub 的是模板文件：
- `config.example.js` - 配置模板（供其他人参考）

### 3. 克隆后使用

别人克隆项目后：
1. 复制 `config.example.js` 为 `config.js`
2. 填入自己的 Supabase 和飞书配置
3. 导入微信开发者工具即可使用

---

## 让朋友使用小程序

### 方式一：体验版（推荐）

不需要审核，快速分享给朋友测试：

1. **在微信开发者工具中点击「上传」**
   - 填写版本号（如：1.0.0）
   - 填写项目备注

2. **登录微信公众平台**
   - 网址：https://mp.weixin.qq.com
   - 用你的小程序账号登录

3. **进入「版本管理」**
   - 找到「开发版本」
   - 点击「提交审核」旁边的 **「体验版」**
   - 生成体验二维码

4. **分享二维码给朋友**
   - 朋友扫码即可使用
   - 不需要审核，立即生效

### 方式二：正式发布

需要微信审核（1-3天）：

1. 提交审核
2. 等待审核通过
3. 点击「发布」
4. 所有微信用户可搜索使用

### 方式三：添加项目成员

如果是长期合作的朋友：

1. 登录微信公众平台
2. 进入「成员管理」
3. 添加朋友的微信号为「体验成员」
4. 朋友可以在微信中直接使用

---

## 注意事项

- **个人主体小程序**：只能添加 15 个体验成员
- **体验版有效期**：30 天，需要定期重新生成
- **审核要求**：确保页面内容合规，没有敏感信息

---

## 多人协作开发

如果朋友也要参与开发：

1. 在 GitHub 添加他为 Collaborator
2. 他克隆代码后，用自己的 config.js
3. 使用自己的小程序 AppID 测试
4. 代码合并到主分支后统一发布
